# Final Project CS361
This is a project for CS-361. This project will be worked on all 
throughout this term (Winter 2026).

This projects name is 'The Mysterious Land of Ur'
and it is a short choose your own adventure game implemented 
primarily with micro services written in Python, c++ and a bit of Java.

This game will be operational with no web but will at a later date
by programmer will be web applicable.
